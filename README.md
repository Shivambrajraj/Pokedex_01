# Pokedex_01


Description:-

This project is a Pokédex web application built using HTML, CSS, and JavaScript.
It allows users to search for Pokémon and view their details such as name, type, and image in an interactive and responsive interface. The project uses an external API to fetch real-time Pokémon data.


Features:-

Search Pokémon by name.
Display Pokémon image and basic details.
Responsive design for mobile and desktop.
Clean and interactive user interface.
Modal popup for Pokémon details.



How to Run the Project:-

Download or clone this repository
Open the index.html file in any modern web browser.
all files are must be present in same folder.




This project helped me strengthen my front-end development skills and understand how real-world APIs are integrated into web applications.
